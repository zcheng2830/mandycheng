Design Context

Function through
a creative lens.

This project was designed at Columbia as a narrative UX exercise — how do you teach nutrition to someone who's never cooked? The answer wasn't a recipe database. It was a guided experience: safety first, ingredients second, cooking third.

Every screen was designed to reduce cognitive load. Flip cards instead of tables. Drag-and-drop instead of instructions. A quiz that reinforces, not tests. The visual language borrows from gamified learning apps but strips away the noise — letting the content breathe.

Built with Flask, vanilla JS, and a lot of user testing.
