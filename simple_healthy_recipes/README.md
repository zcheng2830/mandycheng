Simple Healthy Recipes
Welcome to the Simple Healthy Recipes project! This repository showcases a collection of easy-to-make, nutritious recipes designed for individuals seeking quick and healthy meal options. Whether you're a beginner in the kitchen or an experienced cook, these recipes are crafted to fit seamlessly into your busy lifestyle.

How to Run the Project Locally
To view and interact with the recipes on your local machine, follow these steps:

1. Clone the Repository
git clone https://github.com/rachely2407/simple_healthy_recipes.git
cd simple_healthy_recipes
2. Open the Project
3. Run the project on your web browser
   python3 server.py
